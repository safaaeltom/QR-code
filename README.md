### What I Learned

* How to center elements using Flexbox (`justify-content` and `align-items`)
* Using `gap` instead of margins for consistent spacing
* Structuring a card layout cleanly with padding and border-radius
* Matching Figma spacing and typography (line-height, font sizes)

### Continued Development

* Practice responsive layouts for different screen sizes
* Get more comfortable with typography details (spacing, readability)
